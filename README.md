# FreshCart | E-commerce


## Features

### Authentication
- User Registration
- Login
- Forgot Password
- Reset Password
- Profile Management

### Products & Categories
- Browse Products
- Product Details
- Browse Brands
- Brand Details
- Browse Categories
- Category Details

### Cart & Wishlist
- Add, Remove, Update Products in Cart
- View Cart Summary
- Add & Remove Items from Wishlist

### Checkout & Payment
- Billing Address Form
- Select Payment Method (Cash / Online)
- Place Order
- Checkout Success Page

### Orders
- View Order History
- Order Details

### Additional Pages
- Home Page
- About Page
- Contact Page
- Privacy Policy
- Terms & Conditions
- Not Found Page (404)

## Dependencies
- **React & Next.js** – Core frontend framework
- **TailwindCSS** – Styling & responsiveness
- **Redux** – Client state management
- **React Query** – Server state management
- **React Hook Form & Zod** – Forms & validation
- **FontAwesome & Lucide React** – Icons
- **React Toastify** – Toast notifications
- **Swiper** – Sliders and carousels
- **Axios** – API requests

## Project Structure
- `src/app` – Pages & Routing  
- `src/components` – Shared and auth components  
- `src/features` – Feature-based components and logic  
- `src/hooks` – Custom hooks for auth, cart, wishlist, etc.  
- `src/store` – Redux store & slices  
- `src/assets` – Images and static assets  
- `src/styles` – Global styles
